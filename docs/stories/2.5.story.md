## Story

**Como** administrador do sistema de artigos SEO,
**Quero** que o SEO Scorer verifique se o artigo contem links para fontes autoritativas externas,
**Para que** o Google perceba o artigo como bem fundamentado e confiavel.

---

## Acceptance Criteria

- [ ] AC1: Check "outbound_links" verifica presenca de links externos no artigo
- [ ] AC2: Identifica links para dominios autoritativos (ex: .gov, .edu, pubmed, wikipedia, who.int)
- [ ] AC3: Scoring: 2+ links autoritativos = 5 pts, 1 link = 3 pts, 0 = 0 pts
- [ ] AC4: Nao conta links de CTA (WhatsApp, social media) como outbound autoritativo
- [ ] AC5: Testes unitarios
- [ ] AC6: Zero regressao

---

## Dev Notes

### Abordagem Tecnica

- Extrair todos `<a href="...">` do HTML
- Filtrar links internos (mesmo dominio) e links de CTA (wa.me, instagram, etc.)
- Verificar se restantes apontam para dominios autoritativos
- Lista de dominios/TLDs autoritativos: .gov, .gov.br, .edu, .edu.br, pubmed.ncbi, scholar.google, wikipedia.org, who.int, scielo.br
- Tambem atualizar prompt do Writer para instruir inclusao de 1-2 fontes

### Arquivos
- Modificar: `core/agents/seo_scorer.py`
- Modificar: `config/tenants/_default/prompts/writer.txt` (instruir sobre fontes externas)
