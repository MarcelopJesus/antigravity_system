## Story

**Como** administrador do sistema de artigos SEO,
**Quero** que o SEO Scorer verifique se o artigo menciona entidades relevantes ao topico,
**Para que** o Google entenda melhor sobre O QUE o artigo trata.

## Dev Notes

### Abordagem Tecnica

**Simplificacao (sem API externa):**
- Em vez de usar Google Knowledge Graph API (requer key), usar entidades que o Analyst ja pode gerar
- Adicionar campo `expected_entities` no outline do Analyst (pessoas, conceitos, locais)
- SEO Scorer verifica quantas aparecem no texto
- Scoring: >= 70% presentes = 5 pts, 40-69% = 3 pts, < 40% = 0 pts

**Alternativa futura:** Integrar Google Knowledge Graph API na Fase 6 (6.4 Semantic Entity Mapping)

### Arquivos

- Modificar: `core/agents/seo_scorer.py`
- Modificar: `config/tenants/_default/prompts/analyst.txt` (adicionar campo entities)
