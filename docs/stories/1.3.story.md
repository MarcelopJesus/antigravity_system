# Story 1.3 — Duplicacao Visual no Tema WordPress

**Epic:** 1 - Correcoes Criticas SEO

**Agent:** @dev (Dex)

## Story

**Como** administrador do sistema de artigos SEO,
**Quero** que o artigo publicado no WordPress nao tenha duplicacao visual (titulo + excerpt repetidos antes do conteudo),
**Para que** a experiencia do usuario seja limpa e o SEO nao seja prejudicado por conteudo duplicado.

---

## Problema

O tema WordPress do mjesus.com.br mostra titulo + excerpt ANTES do conteudo do artigo, criando duplicacao visual. O excerpt ja usa meta_description (fix anterior), mas o tema pode ter template customizado que exibe o excerpt.

---

## Acceptance Criteria

- [ ] AC1: Artigo publicado no mjesus.com.br nao mostra conteudo duplicado visualmente
- [ ] AC2: Titulo aparece apenas 1 vez na pagina
- [ ] AC3: Excerpt/meta description nao aparece como texto visivel antes do conteudo
- [ ] AC4: A solucao nao quebra outros artigos ja publicados
- [ ] AC5: Testes existentes continuam passando (zero regressao)

---

## Dev Notes

### Abordagem Tecnica

**Investigacao necessaria (3 opcoes em ordem de preferencia):**

1. **Enviar excerpt vazio:** `wordpress_client.py` envia `excerpt: ""` — mais simples, testa primeiro
2. **CSS no tema:** Adicionar CSS para `display:none` no excerpt do single post template
3. **Ajustar tema:** Modificar `single.php` para nao exibir excerpt no template

### Arquivos Afetados
- `core/wordpress_client.py` — Pode precisar ajustar o envio de excerpt
- Tema WordPress do mjesus.com.br (investigar remotamente)

### Contexto
- Fix anterior: excerpt ja usa meta_description em vez de auto-gerar do conteudo
- O Yoast usa meta_description para SEO (snippet no Google), entao o excerpt vazio nao afeta SEO
- A duplicacao e puramente visual no tema
