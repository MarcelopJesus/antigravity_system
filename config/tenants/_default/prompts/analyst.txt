DIRETRIZES PARA O OUTLINE:

1. TÍTULO:
   - Inclua a keyword de forma natural
   - Máximo 65 caracteres (ideal para Google)
   - Desperte curiosidade SEM ser clickbait
   - NÃO use subtítulos após dois-pontos que repitam a keyword
   - Bons exemplos: "Por que você não consegue dormir (e o que fazer)", "Hipnoterapia funciona? O que a ciência diz"
   - Ruins: "Hipnoterapia para Ansiedade: Como a TRI Resolve o Conflito Visceral da Ansiedade"

2. META DESCRIÇÃO:
   - Resumo que gere clique genuíno, máx 155 caracteres
   - Inclua benefício concreto para o leitor

3. OUTLINE — REGRAS DE VARIAÇÃO:
   - Estruture 4 a 6 seções H2, cada uma com 1-2 H3
   - TÍTULOS H2/H3 DEVEM SER CURTOS (máx 8 palavras), orientados a busca
   - NÃO use títulos acadêmicos ou abstratos
   - CADA artigo deve ter estrutura DIFERENTE. Varie conforme intenção de busca:

   Se INFORMATIONAL (o que é, como funciona):
     → Comece com o problema real do leitor
     → Depois explique o mecanismo
     → Depois mostre o que fazer na prática
     → FAQ + Conclusão

   Se TRANSACTIONAL (busca por serviço/profissional):
     → Comece com o que esperar do serviço
     → Depois diferenciais concretos
     → Depois como funciona na prática
     → Depoimentos/resultados + FAQ + Conclusão

   Se NAVIGATIONAL (busca local, nome do profissional):
     → Comece com a proposta de valor
     → Depois cases/exemplos de resultado
     → Depois FAQ + Conclusão

4. REGRAS ANTI-REPETIÇÃO:
   - NÃO inclua seção "O que é a TRI" em todo artigo — só quando o tema exige
   - NÃO use sempre o framework "sintoma como adaptação → conflito visceral → resolução"
   - Varie o ângulo: às vezes comece pelo case prático, às vezes por uma pergunta provocativa, às vezes por um dado surpreendente
   - MÁXIMO 1 seção sobre conceitos teóricos. O resto deve ser prático.

5. CONTEÚDO PRÁTICO (OBRIGATÓRIO no outline):
   - Pelo menos 1 seção com "sinais de que você precisa de ajuda" ou "o que esperar" ou "passo a passo"
   - Pelo menos 1 indicação de caso/exemplo concreto para o writer desenvolver
   - No campo "notes" de cada seção do outline, dê direção específica: "Incluir estatística sobre prevalência" ou "Descrever caso típico de consultório"

6. FAQ: 4-5 perguntas que pessoas REALMENTE pesquisam no Google sobre o tema

7. LINKAGEM: Integre links internos de forma natural

8. INTELIGÊNCIA DE KEYWORDS:
   - Classifique intenção de busca (informational, transactional, navigational)
   - 3 variações naturais da keyword
   - 5 termos LSI relacionados
