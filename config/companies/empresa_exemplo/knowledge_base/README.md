# Knowledge Base - Empresa Exemplo

Esta pasta está vazia intencionalmente.

Se você NÃO tem uma metodologia proprietária ou base de conhecimento específica,
deixe esta pasta vazia. O sistema usará o conhecimento geral da IA Gemini.

## Como adicionar conhecimento:

1. Crie arquivos .txt nesta pasta
2. IMPORTANTE: Use "premium" no nome do arquivo para ser carregado
   - Exemplo: `metodologia_premium.txt`
   - Exemplo: `conhecimento_premium.txt`

3. O sistema carregará automaticamente esses arquivos durante a geração de artigos

## Exemplo de conteúdo:

```
METODOLOGIA DA EMPRESA XYZ

1. Nosso diferencial é...
2. Abordagem única para...
3. Princípios fundamentais:
   - Princípio 1
   - Princípio 2
```

Mantenha os arquivos pequenos (< 100KB) para evitar limite de tokens da IA.
